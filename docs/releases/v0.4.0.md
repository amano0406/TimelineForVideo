# TimelineForVideo v0.4.0 Tech Preview

Local-first video-to-markdown packaging tool for LLM workflows.
This is a desktop-style local tool, not a hosted SaaS product.

## Baseline Support

- Windows
- Docker Desktop
- CPU mode

## Optional Features

- GPU mode: NVIDIA + Docker GPU access only, best-effort
- Speaker diarization: optional, requires both a Hugging Face token and gated approval for `pyannote/speaker-diarization-community-1`

## Download

- Windows: `TimelineForVideo-windows-local.zip`
- macOS: source-based, experimental path

## What's New

- the product name is now consistently `TimelineForVideo` across the app UI, docs, release bundle, GitHub repository, .NET project, and worker package
- existing installs keep working through migration compatibility for legacy Docker project data, legacy catalog paths, and older environment variable names
- launcher, stop, and uninstall scripts now recognize older `video2timeline` Docker data and reuse or clean it safely during the rename transition

## Known Limitations

- first run downloads models and takes time
- macOS is experimental
- GPU is not the baseline support path
- GUI is the primary supported path
- Docker Desktop is required
